#**Responsive Landing Page**

---

Overview

A responsive landing page built as part of the SkillCraft Technology web development internship (Task 1). The page features an interactive, fixed navigation menu that changes its appearance when the user scrolls or hovers over a menu item, along with a clean, mobile-friendly layout.

Features


Fixed navigation bar, visible at all scroll positions
Navbar background/color changes dynamically on scroll
Hover effects on menu items for better interactivity
Fully responsive layout that adapts to desktop, tablet, and mobile screens


Technologies Used


HTML5
CSS3
JavaScript (DOM manipulation, scroll event handling)


How to Run


Clone this repository: git clone <repo-link>
Open index.html in your browser, or use the Live Server extension in VS Code.
